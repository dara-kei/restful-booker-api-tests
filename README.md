API Tests for Restful Booker

This project contains automated API tests for the Restful Booker application:  
https://restful-booker.herokuapp.com/

About the project

I created API tests to check the main functionality of the booking system.

The tests cover:

- Get token (authentication)
- Get all bookings
- Get one booking by ID
- Create booking
- Update booking
- Delete booking

 What is tested:

- Response status codes
- Response body data
- Correct work of API endpoints
- Authentication process

Technologies

- Python
- Pytest
- Requests

  The goal of this project is to practice API testing and improve my QA skills.
